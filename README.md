# Flower API 

| Endpoint           | Http          | Description  |
| -------------      |:-------------:| ------------:|
| /                  | GET           | Home         |
| /index             | GET           | Flower Info  |
| /index/add         | POST          | Add Flower   |
| /index/delete      | POST          | Delete Flower|


## Stacks
- MongoDB
- Express.js
- Nodemon
- Mongoose
- cors
